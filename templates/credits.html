出典・利用データについて

本アプリケーション「航法計画作成支援システム」では、 航法計画および表示にあたり以下の外部データ・サービスを利用しています。

空域・空港・無線航法施設データ

空域（ASP）、空港（APT）、無線施設（NAV）の位置情報および属性情報は、 OpenAIP が提供するオープンデータを基に整備・加工して使用しています。

  • OpenAIP オープンデータ
    Website: https://www.openaip.net

航空気象（METAR／フライトルール区分）

各空港の METAR 情報および VFR / MVFR / IFR などの飛行カテゴリ（flight rules）は、 AVWX REST API を通じて取得しています。

  • AVWX REST API
    Website: https://avwx.rest

数値予報データ（風向・風速）

巡航高度に最も近い等圧面の風向・風速データは、 Open-Meteo API を通じて GFS 数値予報モデルに基づく予報値を取得し、 航法ログ各レグの風として利用しています。

  • Open-Meteo API（GFS モデル）
    Website: https://open-meteo.com

レーダーエコー

地図上のレーダーエコー表示には、 RainViewer が提供する気象レーダー情報を利用しています。

  • RainViewer Weather Radar API
    Website: https://www.rainviewer.com/

地磁気偏角データ

真方位から磁方位への変換（磁気偏差：VAR）の算出には、 国土地理院が公表する地磁気データを基に作成した偏角グリッドを利用しています。

  • 国土地理院 地磁気データ
    Website: https://www.gsi.go.jp/
